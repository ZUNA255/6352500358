ROS Master คือ ศูนย์รวม Server ในการสื่อสารระหว่าง Node ซึ่งสามารถสร้าง Node ต่างๆได้เพื่อมาดึงข้อมูลในการแลกเปลี่ยนกัน คำสั่งในการเปิดใช้งานคือ roscore 
ros topic คือ เป็นชื่อหัวข้อสื่อกลางที่เราสร้างไว้เพื่อให้สามารถสื่อสารกันได้ 
ROS Service เลยทำหน้าที่นี้ คือ การส่ง Message แบบ ไม่เน้นความต่อเนื่อง มี Request มี Response เช่น ถ้ามีการขอข้อมูล ถึงค่อยส่งไปตามคำสั่ง
ความแตกต่างของ topic กับ message คือ topic เป็นหัวข้อที่เป็นสื่อกลางในการสื่อสารข้อมูล แต่ Message คือ ข้อมูลสารท่ต้องการส่ง เช่น วิดีโอ คือ Message แต่ช่องยูทูป คือ Topic
ความแตกต่างของ Ros topic กับ Ros service คือ ros topic ใครรู้ชื่อ topic ก็สามารถส่งและรับข้อมูลได้ แต่ ros service คือถ้ามีการขอข้อมูลถึงจะส่งไป จะไม่ส่งตลอดเวลาแบบ ros topic
